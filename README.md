
# Salesforce Order Subscription System

## Overview
This project is a Salesforce-based Order Subscription System designed to automate the management of customer orders and invoices. It includes custom objects, flows, and Apex triggers to ensure data integrity, automation, and seamless process management.

## Features
- **Custom Objects**: Order__c, Invoice__c
- **Custom Fields**: Status, Total Amount, Invoice Date
- **Automation**: Flows and Apex triggers for updating order and invoice statuses
- **Permissions**: Role-based access control using Permission Sets

## Tech Stack
- Salesforce (Declarative + Apex)
- Salesforce Flow Builder
- Salesforce DX & Git for version control
